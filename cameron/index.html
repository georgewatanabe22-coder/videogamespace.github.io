Game Overview

Obstacle Escape is a simple survival game based on my last project. In this game, the player moves around to avoid obstacles and stay alive as long as possible. The goal is to react quickly and not get hit.

This game builds on what I made before by improving the movement and making the obstacles more consistent. It uses simple controls and keeps the gameplay easy to understand, but still challenging so players want to keep trying and get better each time.

Game Features

  • Simple obstacle-avoidance gameplay
  • Easy-to-understand objective and controls
  • Player movement using directional controls
  • Collision detection that ends the game on contact
  • Obstacle reset system for continuous play
  • Short start delay to prepare the player before gameplay begins
  • Fast-paced survival challenge
  • Replayable design that encourages improvement
  • Casual game structure that works well for mobile players
  • Clean gameplay loop based on repetition and timing
  • Beginner-friendly design and development approach
  • Built to show how simple mechanics can still create player engagement

Game Pitch

This web page includes several ideas from my original mobile game pitch. The game concept centers on the player-controlled character avoiding moving obstacles in a survival-based format. The design keeps the gameplay simple so it is easier to build, test, and understand.

  • Game Concept: A simple mobile survival game focused on avoiding danger.
  • Target Audience: Casual players who enjoy fast and easy-to-learn games.
  • Development Tools: Created as a prototype using beginner-friendly game development methods.
  • Assets Used: Basic player object, obstacle object, and simple visual elements for gameplay testing.

About the Developer

My name is Cameron Sanchez, and I created this game for my Project 3 Game Web Page assignment. The goal of this project was to make a simple but fun game. I used basic features like player movement, moving obstacles, and collision detection to build the game.

Through this project, I learned that even simple game ideas can still be challenging and fun. I saw how players can keep trying again and improve each time they play. I wanted to make a game that is easy to understand but still fun to play.
